This site covers topics including Java, Spring Boot, Micronaut, Netflix DGS (GraphQL), REST APIs, Docker, microservices, JUnit 5, and more. It is built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Tech Stack

- **Static Site Generator:** Jekyll
- **Hosting:** GitHub Pages
- **Styling:** SCSS / Custom CSS
- **Markdown:** Kramdown
- **Syntax Highlighting:** Rouge
- **Pagination:** jekyll-paginate-v2
- **Feed:** jekyll-feed

---

## Local Development

### Prerequisites

- Ruby (with Bundler)
- Jekyll

### Setup

```bash
# Install dependencies
bundle install

# Serve locally
bundle exec jekyll serve
```

The site will be available at `http://localhost:4000`.

## Project Structure

```
_config.yml        # Site configuration
_layouts/          # Page and post layout templates
_includes/         # Reusable HTML partials (header, footer, etc.)
_posts/            # Blog posts (blog/ and project/ categories)
_sass/             # SCSS stylesheets
css/               # Main stylesheet entry point
images/            # Static images (certifications, etc.)
resources/         # Downloadable files (resume, etc.)
```
